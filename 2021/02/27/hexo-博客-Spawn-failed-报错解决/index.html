hexo 博客 Spawn failed 报错解决

发表于 2021-02-27

报错

解决方法

  1. ssh-keygen -t rsa -C “邮箱地址”
  2. 复制 C:\Users\计算机名称.ssh 中的 id_rsa (id_rsa是你这台电脑的私人秘钥)
  3. 在GitHub的settings下找到SSH and GPG keys，选择New SSH key,将id_rsa.pub 复制进去
  4. 重新部署 hexo d
